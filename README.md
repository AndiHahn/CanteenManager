# CanteenManager

Android Application to manage a Canteen.
Uses CanteenCheckerService.Api: https://canteenchecker.azurewebsites.net/swagger/ui/index#/Admin

- Android Material Design
- Input validation
- Included Google Map
- REST call's
- Recycler View with swipe to delete
- Swipe to Refreshlayout
- Tablayout (with swipe)
- Firebase Cloud Messaging (Push messages)
